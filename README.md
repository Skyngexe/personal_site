# Personal Site

This is a static personal branding site hosted on GitHub Pages. It includes sections for writing, coding projects, and an about page.

## How to Use

1. Clone this repository to your local machine.
2. Make any changes to the `index.html` or `styles.css` files as needed.
3. Push your changes to the `main` branch.
4. Enable GitHub Pages in the repository settings, selecting the `main` branch as the source.

## Structure

- `index.html`: The main HTML file for the site.
- `styles.css`: The CSS file for styling the site.
- `.nojekyll`: Disables Jekyll processing for GitHub Pages.
